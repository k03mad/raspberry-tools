Get data from Raspberry Pi 3

**/help** the list of all available commands  
  
**/arp** get local network devices list  
**/reboot** reboot  
**/shutdown** shutdown  
**/stats** get some usage stats  
**/updates** check for updates  
**/wifi** get hotspots list
